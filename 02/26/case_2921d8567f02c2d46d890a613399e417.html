【裁判日期】 960227
【裁判案由】 詐欺
【裁判全文】   
臺灣高等法院臺中分院刑事判決　　　　95年度上易字第1404號
上　訴　人
即　被　告　甲○○
上列上訴人因詐欺案件，不服臺灣臺中地方法院95年度易字第
1050號中華民國95年9月14 日第一審判決（起訴案號：臺灣臺北
地方法院檢察署93年度偵緝字第1260號），提起上訴，本院判決
如下：
    主  文
上訴駁回。
    事  實
一、甲○○前於民國84年間因贓物罪，經臺灣板橋地方法院判處
    有期徒刑4月，如易科罰金，以300元折算1日確定，於85年
    5月2日易科罰金執行完畢，猶不知警惕。復於87年間，基於
    意圖為自己不法之所有之概括犯意，連續多次冒稱其為旺宏
    企業有限公司（下稱旺宏公司）之業務員，並以「賴建宏」
    之名義，先於87年底及88年初，親自前往臺中縣潭子鄉○○
    村○○○路「大富皮革製品有限公司」（下稱大富公司），
    向該公司實際負責人蔡文啟，以現金新臺幣（下同）7000元
    之代價，購買手機皮套樣品，以取得大富公司之信任。繼而
    自88年2月23日起至4 月5 日止，連續多次以傳真訂貨單之
    方式，向大富公司訂購手機皮套等產品，總計價值260,240
    元，大富公司不知有詐而陷於錯誤，陸續將前開貨物送至甲
    ○○所虛設之公司所在地臺北縣蘆洲市○○街43號2樓；甲
    ○○於88年4月5日最後一次取貨得逞後，即行蹤不明，大富
    公司索討債款無門，始知受騙。
二、案經告訴人大富公司訴由臺灣臺北地方法院檢察署檢察官偵
    查起訴，並經臺灣臺北地方法院94年度易字第906號判決諭
    知管轄錯誤而移送臺灣臺中地方法院審理。
    理  由
一、按被告以外之人於審判外之陳述，雖不符刑事訴訟法第159
    條之1至第159條之4之規定，然經當事人於審判程序同意作
    為證據，法院審酌該言詞陳述作成時之情況，認為適當者，
    亦得為證據；當事人、代理人或辯護人於法院調查證據時，
    知有刑事訴訟法第159條第1項不得為證據之情形，而未於言
    詞辯論終結前聲明異議者，視為有前項之同意，刑事訴訟法
    第159條之5定有明文。查告訴代理人蔡文啟於偵查中所為之
    陳述，固為審判外之陳述而屬傳聞證據，惟被告就前開審判
    外之陳述，於原審及本院準備程序中均不爭執其證據能力，
    本院審酌上開陳述作成時之情況，並無違法取證之瑕疵，認
    以之作為證據為適當，揆諸上開規定，上開審判外之陳述有
    證據能力。
二、訊據上訴人即被告（下稱被告）甲○○矢口否認有詐欺取財
    犯行，於原審及本院準備程序中辯稱伊有向告訴人大富公司
    訂購並收受貨品，惟有給付貨款，而因貨品有瑕疵及規格不
    符，故貨款不能給付，被告確有給付2筆貨款云云。經查上
    開有關被告甲○○連續詐欺取財之犯罪事實，有下列事證可
    資證明：
  (一)告訴人代理人蔡文啟於偵查中及原審審理時指述明確；又於
    臺灣臺北地方法院審理時具結證稱「（問：被告甲○○有無
    以旺宏公司名義向你們公司定貨？）有」，「訂貨金額是
    448,710元，我有帶來對帳資料」，「陸陸續續訂貨，是用
    傳真的」、「一般都是用傳真的」，「（問：如何交貨？）
    被告自己設一個公司，我們把貨寄到貨運公司送到該處去」
    ，「（問：你有無到蘆洲地點查看過？）出事後我有去看，
    裡面是空虛的，貨都拿光了」，「（問：貨款如何跟被告收
    的？）我們出貨，月結3個月，例如8月出貨，9月請款，可
    在11月就會領到錢，但交易1次就出問題了」，「（問：
    被告訂貨時，你有無跟他請款？）有，請了1次，但後來就
    跳票了」，「（跳票後，有無去找被告？）答：有，但找不
    到人」，「（問：被告在訂貨期間有無反應出貨有瑕疵要退
    還貨？）沒有」，「（問：被告以旺宏公司向你們購貨，所
    有購貨的款項都沒有兌現？）是」等語（見94年9月8日審判
    筆錄）。
  (二)證人林昭潃於臺灣臺北地方法院審理時具結證稱：「（問：
    你們有無授權甲○○以中華和記公司名義向大富公司購貨？
    ）沒有」，「（問：你是否認識王農凱？）他曾經是我員工
    ，是被告（指甲○○）的下屬，後來王農凱被被告帶離職了
    ，詳細離職時間我不清楚，我們曾經告過被告，告他把我們
    的貨款侵占，結果法院有判他刑」，「（問：當時已經改名
    ？）當時身分證上是甲○○，後來改名賴建宏」，「（問：
    甲○○當時在你們公司做何事？）經理，負責賣皮套、帶員
    工，他是在桃園分公司任職」，「（問：王農凱做何事？）
    業務，是被告帶的」，「是在我們公司財務出問題前離開的
    」、「（問：離職原因？）他將我們帳款侵占」等語（見94
    年9月8日審判筆錄）。
  (三)證人謝博全於臺灣臺北地方法院審理時具結證稱「（問：被
    告從何時開始任職、至何時離職？）86年底開始在我們公司
    任職，在87年9月停職停薪，88年3月底，我們正式開除他」
    ，「（問：是否認識大富公司？）不認識，我們批盤都是在
    北縣」，「（問：是否認識蔡文啟？）我沒有見過他」，「
    （問：你們公司有無指示被告甲○○向大富公司購買手機皮
    套？）沒有，我沒有聽過大富公司」，「（問：你有無聽過
    手機皮套銷售業務的旺宏公司？）沒有聽過」，「（問：被
    告停職原因？）侵占公司貨款」，「（問：你們如何發現侵
    占貨款？）發現客戶款項沒有回來，我們就去調查」，「（
    問：王農凱是否甲○○下屬？）在臺北時，王農凱不是屬於
    甲○○下屬，在桃園的時候也不是」等語（見94年9月8日審
    判筆錄）。
  (四)證人張廷州於臺灣臺北地方法院具結證稱「（問：你88年間
    在何處工作？）我是在大富皮革公司工作」，「幫助老闆做
    業務上行政工作，將單據下給工廠，幫忙材料出貨到大陸去
    」，「（問：是否認識在庭被告甲○○？指認）沒有什麼印
    象」，「（問：關於旺宏公司與大富皮革公司訂貨的時候，
    代表旺宏公司出面來與大富皮革公司洽談的是何人？）有一
    位先生，我不記得他的名字了」，「（問：該位先生是否是
    訂購單上面的賴建宏？）他來的時候出示如手寫單上的名字
    ，他說他是賴建宏」，「（問：這位賴建宏是否是在庭被告
    ？指認）好像比較瘦一點，我確定就是當時的賴建宏」，「
    （問：旺宏公司有無人曾經向你反應貨出的超過訂購單所要
    求，或是出貨產品有瑕疵、型號不符的情形？）我沒有接過
    這樣的反應」，「（問：當你負責的客戶因為你們公司出貨
    瑕疵，你們如何處理？）就是把東西寄回來，我們再寄新的
    給他」，「（問：你與旺宏公司有接洽時，你是否都是與賴
    建宏聯絡？）是，我與賴建宏只有聯絡過2次而已，我是找
    他名片與他聯絡」，「（提示訂購單並告以要旨）（問：這
    14張都是甲○○向你們公司訂購的訂購單嗎？）是，這是直
    接傳真過來，老闆交給我的」等語（見94年10月20日審判筆
    錄）。
  (五)證人王農凱於臺灣臺北地方法院審理時具結證稱：「（問：
    你當時有無擔任旺宏公司的負責人？）我是合夥人」，「（
    問：旺宏公司其他合夥人是誰？）甲○○」，「（問：你在
    旺宏公司有無實際出資？）沒有，我是出票，票是我個人名
    義的，借給公司使用」，「（問：旺宏公司實際負責人是否
    甲○○？）是」，「（問：旺宏公司需要支票付貨款時，是
    由誰開立的？）甲○○開的」，「（問：你的印章、支票是
    否是放在甲○○處？）是」，「（問：支票兌領時，錢由誰
    去領？）是甲○○」，「（問：你離開旺宏公司後，有無將
    支票借給甲○○？）沒有，當初我覺得款項有點不太對，我
    就將支票收回來」，「4月份收回來」，「（問：你所說款
    項不太對，是指何意？）連續開了3、4張票給客戶，金額蠻
    大的，與一般開出的不太符合，所進的貨都是在倉庫，沒有
    什麼在跑，我當時會擔心，就將票收回」，「（問：你們所
    進的貨在倉庫，倉庫在何處？）蘆洲市○○街我們的辦公處
    所」，「（問：被告將你票開出後，有無退票？）有」，「
    （問：你們進多少貨是由誰決定的？）是甲○○決定的」等
    語（見94年12月21日審判筆錄）。
  (六)被告本名為甲○○，惟於88年2月間係以「賴建宏」之名片
    對外行使，且未於名片中以括號註明本名或可供他人辨識之
    名稱，此有該名片在卷可證，則被告所行使之名片，足致一
    般交易對象無法辨識其真正姓名為甲○○。又依卷附旺宏公
    司公司登記資料顯示，旺宏公司係於77年10月4日成立，77
    年10月14日向主管機關臺灣省政府建設廳申請停業登記，迄
    今並未辦理公司解散登記，則依常情，88年間旺宏公司仍停
    業中，應無雇用被告之可能，則被告焉有可能再行申請設立
    旺宏公司，亦不可能成為旺宏公司之員工。
  (七)綜據上揭證人蔡文啟、林昭滫、謝博全、張廷州、王農凱等
    人證述內容及被告使用之名片、旺宏公司設立登記等資料，
    可知被告甲○○原來任職於中華和記公司，因侵占公司貨款
    後遭該公司開除，旋偽以旺宏公司業務員「賴建宏」之名，
    連續向大富公司施用詐術購買貨品，被告甲○○確實有為自
    己不法所有之意圖甚明。
  (八)此外復有旺宏公司基本資料查詢、變更登記事項卡、訂購單
    、大富公司交易明細表、應收帳款明細表在卷可稽。被告雖
    辯稱其因取得之貨品有瑕疵及型號不符始未付款云云，惟被
    告未將其指稱有瑕疵或型號不符之貨品寄回換新，亦經被告
    於原審審理時自承在卷，則其買受之貨品倘確有瑕疵情形，
    豈可能逕不付款、又不向告訴人退換貨品？被告所辯，顯屬
    卸責之詞，不可採信。本件事證已臻明確，被告犯行堪予認
    定。
三、按刑法及刑法施行法業於94年2月2日總統以華總一義字第
    0940一義字第09400014901號令修正公布，並於95年7月1日
    施行（下稱新刑法，修正前刑法下稱舊刑法），如附表所示
    之條文均有修正，參酌最高法院95年5月23日刑事庭第8次會
    議決議及95年11月7日第21次刑事庭會議95年刑議字第6號決
    議，新刑法第2條第1項之規定，係規範行為後法律變更所生
    新舊法律比較適用之準據法，於新刑法施行後，應適用新刑
    法第2條第1項之規定，為「從舊從輕」之比較。經比較後，
    理由及結果均詳如附表所示。
四、核被告甲○○所為，係犯刑法第339條第1項之詐欺取財罪。
    被告所為多次詐欺犯行，時間緊接，犯罪構成要件相同，顯
    係基於概括犯意所為，應依修正前刑法第56條連續犯之規定
    ，以1罪論，並加重其刑。又被告前於84年間因贓物罪，經
    臺灣板橋地方法院判處有期徒刑4月，如易科罰金，以300
    元折算1日確定，於85年5 月2 日易科罰金執行完畢，有臺
    灣高等法院被告前案記錄表在卷可參，其於受徒刑執行完畢
    後，5年以內故意再犯本件有期徒刑以上之罪，為累犯，應
    依修正前刑法第47條規定，遞予加重其刑。
五、原審以被告罪證明確，因之適用刑法第2條第1 項前段、第
    339條第1項、修正前刑法第56條，修正前刑法第47條、修正
    前刑法第41條第1項前段，罰金罰鍰提高標準條例第1條前段
    、修正前罰金罰鍰提高標準條例第2條等規定，並審酌被告
    犯罪之動機、目的、手段、品行、所生危害及犯罪後態度
    等一切情狀，量處被告有期徒刑6月，並諭知易科罰金之折
    算標準，經核其認事用法及量刑並無違誤之處，應為適法。
    被告提起上訴，猶矢口否認犯罪，非有理由，應予駁回。
六、被告經合法傳喚，無正當理由不到庭，爰不待其陳述逕為判
    決。
據上論斷，應依刑事訴訟法第371條、第368條，判決如主文。
本案經檢察官乙○○到庭執行職務。

│  │  之宣告，因身體、教育、職業、家庭之關係或其他正當│                │
│  │  事由，執行顯有困難者，得以1元以上3元以下折算1日 │                │
│  │  ，易科罰金。」                                  │                │
│  │(三)修正後第41條第1項前段：「犯最重本刑為5年以下有  │                │
│  │  期徒刑以下之刑之罪，而受6個月以下有期徒刑或拘役 │                │
│  │  之宣告者，得以新臺幣1千元、2千元或3千元折算1日  │                │
│  │  ，易科罰金。」                                  │                │
├─┼─────────────────────────┴────────┤
│結│(一)依現行刑法第2條第1項：「行為後法律有變更者，適用行為時之法律。但行│
│　│  為後之法律有利於行為人者，適用最有利於行為人之法律。」之從舊從輕規│
│論│  定，因編號 2、 3、  4   所示修正後刑法對被告並非有利，故仍應依刑法│
│　│　第2條第1項前段之規定，適用修正前之刑法相關規定。                  │
│  │(二)雖原審就編號  1   所示適用修正前刑法規定，惟對被告之處罰結果並無不│
│  │  同，依最高法院95年11月7日第21次刑事庭會議95年刑議字第6號決議，自毋│
│  │  庸以此事由加以撤銷。                                              │
