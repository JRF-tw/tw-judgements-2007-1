臺灣彰化地方法院刑事判決　　　　　　　　96年度易字第33號
公　訴　人　臺灣彰化地方法院檢察署檢察官
被　　　告　己　○
　　　　　　　　　　　號
　　　　　　壬○○
上列被告等因竊盜等案件，經檢察官提起公訴（95年度偵字第85
50號），被告等就被訴事實均為有罪之陳述，經裁定改行簡式審
判程序，本院判決如下：
    主  文
己○犯如附表壹所示之罪，各應處如附表壹所示之刑，並定應執
行有期徒刑貳年。扣案之老虎鉗、破壞剪及美工刀各壹支，及美
工刀刀片貳片均沒收。
壬○○犯如附表貳所示之罪，各應處如附表貳所示之刑，並就拘
役部分，定應執行拘役捌拾日，如易科罰金，以新臺幣壹仟元折
算壹日。均緩刑貳年。
    犯罪事實及理由
一、犯罪事實部分：
    己○前有贓物及多次竊盜前科，素行非佳（均不構成累犯）
    。其因缺錢花用，竟基於意圖為自己不法所有之概括犯意，
    於附表壹編號1 至19所示之時間、地點，持其所有，客觀上
    足對人之生命、身體構成危險，屬兇器之老虎鉗、破壞剪及
    美工刀各1 支，竊取附表壹編號1 至19所示甲○○、丁○○
    、丙○○、癸○○○、辛○○等人所有，如附表壹編號1 至
    19所示之電線財物。己○復另行起意，分別基於意圖為自己
    不法所有之犯意，先後於附表壹編號20至31所示之時間、地
    點，持其所有，客觀上足對人之生命、身體構成危險，屬兇
    器之同上老虎鉗、破壞剪及美工刀各1 支，竊取附表壹編號
    20至31所示乙○○、庚○○、戊○○、辛○○、癸○○○、
    子○○等人所有，如附表壹編號20至31所示之電線財物。己
    ○於竊得前開電線後，於將電線外緣剝除或燒熔後，即將所
    餘銅線載往壬○○所經營，位於彰化縣田中鎮○○里○○街
    26號「順惠舊貨商」（招牌為惠泉舊貨行），將竊得之銅線
    變賣予壬○○。而壬○○係以經營收購舊貨為業，且平日收
    購舊貨時，均會向出售者要求登記年籍資料，詎其竟基於故
    買贓物之概括犯意，明知己○於附表貳編號1 至5 所示時間
    、地點，所兜售如附表貳編號1 至5 所示之銅線，均係來路
    不明之贓物（分係己○於附表壹編號1 至19所示時間、地點
    竊盜所得），竟未要求己○填寫個人基本資料，即以附表貳
    編號1 至5 所示之低價予以故買。壬○○復另行起意，分別
    基於故買贓物之犯意，明知己○於附表貳編號6 至9 所示時
    間、地點，所兜售如附表貳編號6 至9 所示之銅線，均係來
    路不明之贓物（分係己○於附表壹編號20至31所示時間、地
    點竊盜所得），竟仍未要求己○填寫個人基本資料，即以附
    表貳編號6 至9 所示之低價予以故買。嗣於95年8 月28日子
    ○○發現可疑為竊嫌所騎乘之車號IWH —329 號停放在彰化
    縣田尾鄉○○村○○路○ 段455 巷26號前，乃會同乙○○於
    同日12時許，報警前往該機車停放址查訪，經當時居住在該
    址之己○承認曾為附表壹所示之竊盜犯行，且在該址內發現
    一袋剝完銅線之電線塑膠外皮，並扣得己○所有，供其犯附
    表壹所示竊盜案件時，所使用之老虎鉗、破壞剪及美工刀各
    1 支，及其所有，預備供其犯竊盜罪所用之美工刀刀片2 片
    。嗣後經己○帶同員警前往銷贓地點即壬○○所經營之順惠
    舊貨商，而查知壬○○故買贓物犯行，而知悉上情。
二、被告己○、壬○○於本院準備程序中，就上揭犯罪事實均為
    有罪之陳述，且其等所犯均係死刑、無期徒刑、最輕本刑為
    3 年以上有期徒刑以外之罪，經本院合議庭依刑事訴訟法第
    273 條之1 第1 項之規定，裁定由受命法官1 人獨任進行簡
    式審判程序。
三、本案證據：
   (一)證人即被害人甲○○、丁○○、丙○○、辛○○、癸○○
     ○、子○○、乙○○、庚○○及戊○○之證述。
   (二)贓物認領保管單3紙。
   (三)查獲現場照片21紙。
   (四)扣案之老虎鉗、破壞剪及美工刀各1支及美工刀刀片2片。
   (五)被告己○、壬○○之自白。
四、我國刑法於94年2月2日修正公布，95年7月1日施行（下稱新
    法；修正前刑法下稱舊法）後，依最高法院95年度第 8次刑
    事庭會議決議決議內容，有關新舊法之適用原則如下：
  　(一)新法第2條第1項之規定，係規範行為後法律變更所生新舊
      法律比較適用之準據法，於新法施行後，應適用新法第2
      條第1項之規定，為「從舊從輕」之比較。
    (二)基於罪刑法定原則及法律不溯及既往原則，行為之處罰，
      以行為時之法律有明文規定者為限，必行為時與行為後之
      法律均有處罰之規定，始有新法第2條第1項之適用。
    (三)比較時應就罪刑有關之共犯、未遂犯、想像競合犯、牽連
      犯、連續犯、結合犯，以及累犯加重、自首減輕暨其他法
      定加減原因（如身分加減）與加減例等一切情形，綜其全
      部罪刑之結果而為比較。
    (四)新法第33條第5 款規定罰金刑為新臺幣1 千元以上，以百
      元計算之，新法施行後，應依新法第2 條第1 項之規定，
      適用最有利於行為人之法律。
    (五)易科罰金之折算標準，新法施行後，應依新法第2 條第1
      項之規定，適用最有利於行為人之法律。
    (六)新法第51條第5 款提高多數有期徒刑合併應執行之刑不得
      逾30年，新法施行後，應依新法第2 條第1 項之規定，適
      用最有利於行為人之法律。
    (七)犯罪在新法施行前，新法施行後，緩刑之宣告，應適用新
      法第74條之規定。
    (八)從刑附屬於主刑，除法律有特別規定者外，依主刑所適用
      之法律。
    本案被告己○為附表壹編號1 至19所示竊盜行為後，刑法關
    於連續犯、定應執行刑等規定業已修正；被告壬○○於為附
    表貳編號1 至5 所示故買贓物行為後，刑法關於連續犯、主
    刑罰金刑及易科罰金等規定業已修正，依前揭最高法院95年
    度第8 次刑事庭會議決議內容，應綜其全部罪刑之結果而為
    比較後，始得判斷應適用新法或舊法，經本院就被告己○及
    葉志昌分別為如附表、肆之比較後，認本件被告己○所犯
    如附表壹編號1 至19所示之竊盜犯行，應依刑法第2 條第 1
    項前段，就連續犯及定應執行刑之適用，均應依舊法，且依
    從刑附屬於主刑之原則，此部分之沒收，亦應適用舊法第38
    條之規定；被告壬○○所犯如附表貳編號1 至5 所示故買贓
    物犯行，應依刑法第2 條第1 項前段，就連續犯、主刑罰金
    刑及易科罰金部分，亦均應依舊法。
五、應依刑事訴訟法第273條之1第1項、第 299條第1項前段、第
    310 條之2 、第454 條第1 項，刑法第2 條第1 項前段，修
    正刪除前刑法第56條，刑法第321 條第1 項第3 款、第 349
    條第2 項、修正前刑法第41條第1 項前段、第51條第5 款，
    刑法第41條第1 項前段、第51條第6 款、第74條第1 項第 1
    款，修正前刑法第38條第1 項第2 款，刑法第38條第1 項第
    2 款，刑法施行法第1 條之1 第1 項、第2 項前段，罰金罰
    鍰提高標準條例第1 條前段，修正刪除前罰金罰鍰提高標準
    條例第2 條，判決如主文。
六、如不服本判決，應於判決送達後10日內，向本院提出上訴狀
    （須附繕本），上訴於臺灣高等法院臺中分院。
本案經檢察官吳宗達到庭執行職務。

犯竊盜罪而有左列情形之一者，處 6 月以上、 5 年以下有期徒
刑：
一  於夜間侵入住宅或有人居住之建築物、船艦或隱匿其內而犯
之者。
二  毀越門扇、牆垣或其他安全設備而犯之者。
三  攜帶兇器而犯之者。
四  結夥三人以上而犯之者。
五  乘火災、水災或其他災害之際而犯之者。
六  在車站或埠頭而犯之者。
前項之未遂犯罰之。

或科或併科 1 千元以下罰金。

│編號│修正事項  │  以各別修正事項討論新、舊法之比較適用            │
├──┼─────┼─────────────────────────┤
│ 一 │連續犯    │被告壬○○為如附表貳編號1 至5 所示行為後，刑法第56│
│    │          │條連續犯之規定，業經刪除，則被告壬○○之此部分犯行│
│    │          │，因行為後新法業已刪除連續犯之規定，此刪除雖非犯罪│
│    │          │構成要件之變更，但已顯已影響行為人刑罰之法律效果，│
│    │          │自屬法律有變更，且比較新、舊法結果，以舊法論以連續│
│    │          │犯對被告壬○○較為有利                            │
├──┼─────┼─────────────────────────┤
│ 二 │易科罰金  │刑法第41條第1項前段關於得易科罰金之規定，修正前係 │
│    │          │以「因身體、教育、職業、家庭之關係或其他正當事由，│
│    │          │執行顯有困難」為其要件，新法則刪除此要件，修正理由│
│    │          │說明上述規定限制過嚴，欠缺彈性，與易科罰金制度旨在│
│    │          │救濟短期自由刑流弊之旨趣不符，且刪除該要件後使檢察│
│    │          │官得視具體個案審酌宜否易科罰金，更符告易科罰金制度│
│    │          │之意旨。另修正前刑法第41條第1項前段配合罰金罰鍰提 │
│    │          │高標準條例第2條（現已刪除）之規定，係以「銀元300元│
│    │          │即新台幣900元折算1日」為易科罰金之折算標準，至修正│
│    │          │後之刑法第41條第1項乃係以「新台幣1,000元、2,000元 │
│    │          │或3,000元折算1日」為易科罰金之折算標準。而觀諸歷年│
│    │          │來之實務運作情形，在法院宣告易科罰金折算標準之案件│
│    │          │，檢察官執行時乃以准予易科罰金為原則，不准易科罰金│
│    │          │為例外，是以比較修法前後就易科罰金准否及折算標準之│
│    │          │規定，應認修正後刑法第41條第1項前段提高易科罰金之 │
│    │          │折算標準之適用，對於行為人而言較為不利            │
